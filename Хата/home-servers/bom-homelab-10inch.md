## Контрольні перевірки перед оплатою

- [ ] Версія плати CWWK: саме **i5-12450H** (не U300e — у нього 14 ліній замість 28), кіт саме **4×U.2**

- [ ] Кабелі під HBA: **SFF-8087**→SATA forward (не 8643, не reverse)
- [ ] ECC-модулі в Memory QVL AM5D4ID; DAC для NAS — Intel-coded
- [ ] ENP-7660B: активний PFC 200–240В; має EPS 4+4

## Перший запуск NAS (пам'ятка)

1. БЖ з джампером → плата стартує сама (перевірити Restore AC Power Loss = Power On у BIOS).
2. HBA: переконатися, що BIOS бачить option ROM, диски видно в `lsblk`; `smartctl -a` по кожному.
3. badblocks/повний SMART-тест дисків до створення пулу; boot-дзеркало з 2× SATA SSD.
4. 82599: якщо DAC не підніметься — `options ixgbe allow_unsupported_sfp=1` (план Б, з Intel-coded не знадобиться).
5. Scrub-марафон тиждень під навантаженням до заливки реальних даних.

# PDU / POWER-ЮНІТ СТІЙКИ (проектування, червень 2026)

## Концепція
Окремий sled під Lab-RAX (плаский горизонтальний, ~1.5–2U) з трьома фізично розділеними зонами:
**220В силова / комутація (реле) / слабкострум+DC**. Силову частину 220В Сергій робить сам як електрик
(досвід панелі Боровиці), на готових/перевірених компонентах — не саморобні 220В-реле.

## Принцип розподілу (інженерна знахідка)
Критичний домен (DNS/мережа/HA/безпека) виявився переважно DC/PoE або з власним резервом,
а важкі робочі машини — AC. Три домени:
1. **AC керовані** (через реле з/без виміру): NAS, Proxmox + запас
2. **AC некомутовані** (завжди під напругою, окремі розетки без реле): Ajax NVR, Mac mini
3. **DC 48В шина** (від спільного блока): RB5009 + CRS305 + PoE-периферія

## Відкриті питання для механіки (коли почнемо креслити sled)
1. Кількість розеток C13 + розподіл керовані/некомутовані (керовані: NAS, Proxmox + запас; некомут.: NVR, Mac mini)
2. Орієнтація вводу 220В (назад/вбік), один ввід чи кілька
3. Глибина sled-а → визначає DC-блок (UHP vs LRS-150F) і чи влазить 199мм
4. Монтаж реле → визначає вибір Shelly(DIN) vs 8CH(стійки)

# КОРПУСИ 3D-ДРУК — ОСНОВА Lab-RAX (проектування)

## База
**Lab-RAX** від mklements (makerworld 1294480, bolted-версія 1464819 — без втулок, M6/#10-32 болти).
Готова модульна рама (рама+пости+панелі), 20.9k завантажень. Колекція аксесуарів:
makerworld collections 6647137 (depth-spacers, fan-mounts тощо).
Проектуємо НЕ корпуси з нуля, а **sled-вставки** під готовий стандарт Lab-RAX.

## Розміри 10" стійки (канонічні, EIA-310)
- Зовнішня ширина фронту: **254 мм** (10")
- **Корисна ширина між рейками: 222.25 мм** (8.75") — головне обмеження
- Рейки 15.875 мм (0.625"), 1U = 44.45 мм
- Принтер Bambu A1: поле 256мм → панель 254мм ВПРИТУЛ → друк по частинах

## Спільні принципи (фундамент обох корпусів)
1. Рама+знімні панелі (не моноблок); з'єднання M3 у heat-set втулки (розбірність)
2. Орієнтація друку: навантаження ВЗДОВЖ шарів; вуха під рейки — площиною в стіл, 4-5 ліній стінок, 100% інфіл у зонах отворів
3. Матеріал: **PETG** база; **ASA** біля гарячого (VRM/HBA/БЖ); PLA — НІ (попливе)
4. **Повітря — наскрізний принцип:** холодне спереду → крізь гарячі точки → витяжка ззаду; спрямований тунель, не "вентилятор у стінку"
   - NAS: продув КРІЗЬ кейдж дисків + вздовж радіатора HBA
   - Proxmox: продув вздовж VRM/NIC (саме перегрів VRM убив Топтон!)
5. Вентилятори: Arctic P8(80мм)/P12(120мм) PWM (4-pin)

# РОЗШИРЕННЯ СТОРЕДЖУ / PCIe НА PROXMOX (на майбутнє, за потреби)

## Карта ліній AM5D4ID2 (з block diagram -2L+/BCM, платформа ідентична)
28 ліній CPU розподілені фіксовано:
- **PCIE 0–15** → слот PCIEx16 SLOT1 (Gen5 x16), біфурцирується
- **PCIE 16–19** → **M.2_1** (Gen4 x4) — ЄДИНИЙ нативний M.2 на платі!
- **PCIE 20** → BMC/IPMI (Gen3 x1)
- **PCIE 22–23** → **OCuLink OCU1** (Gen4 **x2**, фізичний конектор x4 але електрично x2), НЕЗАЛЕЖНИЙ
- **PCIE 24–27** → бортова мережа (BCM5720/i210)

Біфуркація слота підтверджена в BIOS: **PCIE7 Link Width: x16 / x8x8 / x8x4x4 / x4x4x8 / x4x4x4x4**.
CX4121C електрично потребує лише **x4** (2×10G лізе в x4 Gen4 з запасом), хоч механічно це x8-карта.

## Поточна схема (без рига, $0 додатково)
CX4121C у слот напряму (x16) + M.2_1 (система 990 Pro) + OCuLink x2 (другий диск, через OCuLink→M.2 адаптер ~$20).
= 10G + 2 диски. Третій диск — OCuLink. Це базовий план, ригу не потребує.

(turn 90°, Gen4, екранований, UL). У кожен винесений слот — БУДЬ-ЯКА карта (мережа/диск-адаптер/HBA/GPU).
